# WEBGIS GIÁM SÁT NƯỚC MẶT

Dự án WebGIS giám sát chất lượng nước mặt sử dụng:

- Django
- Leaflet
- Node.js
- JavaScript
- HTML/CSS
- Python

## Chức năng
- Hiển thị bản đồ chất lượng nước
- Nội suy dữ liệu ô nhiễm
- Dashboard thống kê
- Cảnh báo khu vực ô nhiễm

## Cách chạy project

### Backend 
cd backend
node server.js

### Frontend
cd webgis
py manage.py runserver
